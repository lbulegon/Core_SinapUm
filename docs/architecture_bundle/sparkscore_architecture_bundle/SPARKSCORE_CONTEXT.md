# SPARKSCORE CONTEXT

O SparkScore é um orbital de análise psicológica e semiótica dentro do Core_SinapUm.

Seu objetivo é analisar estímulos com base na teoria de Peirce, fornecendo análise completa (SparkScore + PPA + Orbital + Semiótica + Psicológica + Métricas).

Porta: 8006. Framework: FastAPI.
