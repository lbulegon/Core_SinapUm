# SHOPPERBOT ARCHITECTURE MANIFEST

## 1. Identificação

- Sistema: ShopperBot
- Tipo: Orbital especializado
- Ecossistema: Core_SinapUm
- Natureza: IA vendedora VitrineZap

---

## 2. Tese arquitetural

O ShopperBot é um orbital de atendimento conversacional com inteligência de vendas dentro do ecossistema SinapUm.

Sua função é atender conversas, classificar intenções, recomendar produtos e realizar handoff quando necessário.

---

## 3. Objetivo do orbital

O ShopperBot existe para:

- classificar intenções (intent/classify)
- recomendar produtos
- integrar com Creative Engine para conteúdo
- realizar handoff para outros canais
- servir o VitrineZap

---

## 4. Princípios arquiteturais

1. O ShopperBot é um orbital do SinapUm.
2. Integração com Creative Engine e outros serviços.
3. Serviços como tasks síncronas.
4. Deve respeitar a arquitetura do Core_SinapUm.
