# CREATIVE_ENGINE CONTEXT

O Creative Engine é um orbital de geração criativa dentro do Core_SinapUm.

Seu objetivo é fornecer APIs para criação de conteúdo (texto, imagem) usando modelos de IA, integrando-se ao ecossistema de serviços.

Integra com OpenMind e outros provedores de LLM para geração multimodal.
