# TASKS — Sistema de Prompts | Core_SinapUm

Este documento registra TODAS as tarefas que impactam o sistema de prompts do Core_SinapUm. Cada tarefa representa uma alteração, criação ou revisão de um ativo cognitivo (prompt, regra ou governança).

**Nenhuma mudança em prompts ocorre sem uma TASK associada.**

---

## Convenções

- TASKs são identificadas como: `TASK-XXX`
- TASKs sempre impactam uma **versão**
- Uma versão pode conter várias TASKs
- TASKs nunca são apagadas (histórico permanente)

---
